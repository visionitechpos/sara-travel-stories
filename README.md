# Sara Travel Stories

## Overview

Sara Travel Stories is a long-term AI-powered travel storytelling project centered around Sara, a digital Saudi traveler from Jeddah.

The goal is not to create random AI-generated videos.

The goal is to build a consistent digital personality capable of exploring places, sharing stories, documenting culture, and creating authentic travel experiences through high-quality AI-generated content.

Sara should feel like a real person, not an AI character.

---

# Mission

Create a sustainable AI-first travel media brand built around storytelling, exploration, culture, history, and human experiences.

Every episode should:

* Tell an interesting story.
* Teach something valuable.
* Showcase authentic locations.
* Strengthen Sara's identity.
* Feel natural and believable.

---

# Project Vision

The long-term vision is to build a living digital personality capable of:

* Exploring cities.
* Visiting landmarks.
* Sharing local culture.
* Traveling internationally.
* Documenting experiences.
* Growing over time through consistent storytelling.

Sara is the foundation of the project.

Character consistency is more important than individual videos.

---

# Core Identity

Sara is:

* A Saudi woman from Jeddah.
* A travel storyteller.
* Curious and approachable.
* Friendly and authentic.
* Passionate about discovering stories behind places.

Sara is NOT:

* A news presenter.
* A commercial spokesperson.
* A fashion model.
* An AI influencer focused on trends.

Sara's primary role is storytelling.

---

# Production Philosophy

The project follows a simple improvement loop:

Problem
→ Analyze
→ Fix
→ Test
→ Document
→ Repeat

Never change multiple major variables at once.

Small improvements produce long-term consistency.

---

# Golden Rules

## Locked Assets

The following assets must not be changed without explicit approval:

* Sara's face
* Sara's voice
* Sara's age
* Sara's personality
* Sara's core identity

These are considered project foundations.

---

## Language Rules

Sara speaks naturally.

Write for speech, not reading.

Use short conversational sentences.

Avoid:

* Formal Arabic
* Academic language
* News presenter style
* Overly scripted dialogue

The goal is authentic spoken communication.

---

## Voice Rules

Voice consistency is critical.

Do not replace the approved voice.

Improve pronunciation through:

* Better scripts
* Better wording
* Better speech direction

Never solve voice issues by constantly changing voice models.

---

## Environment Rules

Environment consistency matters.

Reference images are preferred over long environment descriptions.

Real locations should be represented as accurately as possible.
