# References

This folder contains reference images for locations, landmarks, environments, and visual direction.

Reference images are the primary source for environment production.
They are always preferred over written descriptions.

---

## Folder Structure

```
references/
  jeddah/
    al-balad/
    corniche/
    general/
  landmarks/
    [landmark-name]/
  international/
    [country]/
      [city]/
```

---

## Where the Actual Images Live

Starting 2026-07-01, actual image files are stored in Google Drive (`visionitechpos@gmail.com`), not in this repo — see `episodes/PRODUCTION_DECISIONS_LOG.md` ("Media Storage Moved to Google Drive"). This repo keeps the folder structure and a `DRIVE_LINK.md` note per location pointing to the corresponding Drive folder.

## How to Use References

1. Before producing any location content, check this folder first (and its `DRIVE_LINK.md` if present).
2. If references exist — use them as the primary visual source.
3. If references do not exist — collect them before production begins, and upload to the matching Drive folder (create one if needed, mirroring this repo's path).
4. Add a `DRIVE_LINK.md` in the corresponding local folder pointing to the Drive location.

---

## File Naming Convention

`[location]-[description]-[number].[ext]`

Examples:
* `al-balad-rawasheen-01.jpg`
* `jeddah-corniche-sunset-01.jpg`
* `istanbul-grand-bazaar-entrance-01.jpg`

---

## Image Standards

* Minimum resolution: 1920×1080
* Format: JPG or PNG
* Source: Real photographs preferred over illustrations
* Rights: Only use images you have rights to use in production
