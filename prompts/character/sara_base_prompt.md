# Sara — Base Character Prompt

## Status: LOCKED

This prompt defines Sara's visual identity.

Do not modify the core character description without explicit approval.

---

## Base Character Description

```
Sara is a Saudi woman in her late 20s from Jeddah. 
She has a warm, natural appearance. Dark hair, brown eyes. 
Her expression is calm and approachable. 
She looks like a real person, not a model or presenter.
She is dressed modestly and practically for travel.
Natural skin, no heavy makeup. Authentic look.
```

---

## Usage

Use this description as the foundation for all visual prompts involving Sara.

Add location context and shot type on top of this base.

### Example Combined Prompt

```
[BASE CHARACTER DESCRIPTION]
Sara is standing in a narrow alley in Al-Balad, Jeddah's historic district.
Old coral-stone buildings with wooden latticed windows surround her.
Warm morning light. She is looking at an old door, curious.
Medium shot. Candid, natural pose. Not staged.
Cinematic quality. Photorealistic.
```

---

## What NOT to Change

* Sara's face structure and features
* Her age (late 20s — not younger, not older)
* Her natural, approachable quality
* The modest, practical clothing style

## What Can Vary Per Episode

* Outfit colors and specific clothing (within modest travel style)
* Hair — up or down, different styles
* Expression — matching the scene's emotional tone
* Location and environment
