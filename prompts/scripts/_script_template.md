# Script Template

## Before You Write

Review these documents first:
- `docs/SARA_LANGUAGE_BIBLE.md` — language rules
- `docs/SARA_VOICE_BIBLE.md` — voice and pacing rules
- Episode `CONCEPT.md` — the story you are writing

---

## Script Header

```
Episode: ep[number]_[location-slug]
Draft: [1 / 2 / Final]
Date: [YYYY-MM-DD]
Status: [ ] Draft | [ ] Revised | [ ] Read Aloud Done | [ ] Final
Estimated duration: [X] minutes
```

---

## Script Format Rules

* Write dialogue only — no stage directions in the final script.
* Use `[pause]`, `[long pause]`, `[warm]`, `[focus]`, `[wonder]` for voice direction.
* Keep all paragraphs short — 2-3 sentences maximum.
* Leave a blank line between each paragraph.

---

## Script Body

```
[OPENING — start in the moment, no introduction]


[PARAGRAPH 2]


[PARAGRAPH 3]


[PARAGRAPH 4]


[PARAGRAPH 5]


[CLOSING — complete the story, do not summarize]
```

---

## Self-Review Checklist

Before submitting this script for production:

- [ ] Every sentence can be said in one breath
- [ ] No sentence sounds like written text
- [ ] Opening does not start with "أهلاً" or a formal greeting
- [ ] Closing leaves something behind — not a summary
- [ ] No news-presenter or formal Arabic phrasing
- [ ] Script was read aloud at least once
- [ ] Word count appropriate for target duration (approx 120-150 words/minute)
