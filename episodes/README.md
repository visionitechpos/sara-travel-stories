# Episodes

This folder contains all episode concepts, scripts, production notes, and reviews.

---

## Folder Structure

Each episode lives in its own subfolder:

```
episodes/
  ep001_al-balad-jeddah/
    CONCEPT.md
    SCRIPT.md
    PRODUCTION_NOTES.md
    REVIEW.md
  ep002_.../
    ...
  _TEMPLATE/
    CONCEPT.md
    SCRIPT.md
    PRODUCTION_NOTES.md
    REVIEW.md
  PRODUCTION_DECISIONS_LOG.md
```

---

## Episode Naming Convention

`ep[number]_[location-slug]`

Examples:
* `ep001_al-balad-jeddah`
* `ep002_istanbul-grand-bazaar`
* `ep003_cairo-old-city`

Always use lowercase and hyphens. No spaces.

---

## Episode Lifecycle

1. **Concept** — Story idea, location, core question
2. **Script** — Full written script, reviewed against Language Bible
3. **Production** — Visuals generated, voice produced, assembled
4. **Review** — Episode review checklist completed
5. **Published** — Episode live, notes filed

Mark the current stage in CONCEPT.md under the Status field.

---

## Production Decisions Log

All significant production decisions across all episodes are logged in:

`PRODUCTION_DECISIONS_LOG.md`

This is the institutional memory of the project.
