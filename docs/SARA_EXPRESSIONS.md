# Sara Expressions

## Purpose

This document is not a Hejazi dialect dictionary. It is not a list of popular sayings.

It is a growing record of expressions, sentences, and phrases that proved themselves to be part of Sara's voice through actual production — not through documentation design.

**The distinction from `SARA_LANGUAGE_BIBLE.md`:**

* `SARA_LANGUAGE_BIBLE.md` describes how Sara *should* speak — vocabulary, sentence structure, forbidden language, speaking style. It is designed knowledge.
* `SARA_EXPRESSIONS.md` documents how Sara *actually spoke* once tested — it is evidence, not design.

This file starts nearly empty. It grows one episode at a time, never all at once.

---

## Entry Criteria

Nothing enters this file because it is Hejazi. Nothing enters because it is famous. An expression earns a place here only if:

1. **It appeared during real production** — a script that was actually taken through generation for an episode, not a hypothetical example.
2. **It was proposed by the AI during dialect refinement**, and held up under review.
3. **It was reviewed and approved** — no expression is added unilaterally without confirmation.

If an expression does not meet one of these three conditions, it does not belong here — even if it sounds authentically Hejazi. Authenticity is necessary but not sufficient. Proof through use is the bar.

---

## How This File Is Used

The Character Cognition Engine draws on this file the same way it draws on `SARA_PERSONA_BIBLE.md`, `SARA_LANGUAGE_BIBLE.md`, and `SARA_VOICE_BIBLE.md` — as a source of what Sara's expression actually sounds like in practice, not only what the rules say it should sound like.

Over time, if an expression here appears consistently enough that it becomes indistinguishable from a core vocabulary rule, that overlap is a signal to revisit organization — but that decision is made later, based on production evidence, not now.

## Entry Format

```
### [Expression or line]

| Field | Value |
|-------|-------|

| **Context of Use** | The specific situation where this was said or where it fits |
| **Why It Earned Its Place** | What made this feel like Sara, specifically |
| **Status** | Adopted / Under Review |
```
