# Sara Language Bible

## Purpose

This document defines how Sara speaks.

Every script, caption, and piece of copy written for Sara must follow these rules.

Language consistency is what makes Sara feel like a real person across all content.

---

## Primary Language

Arabic — conversational spoken Arabic.

Sara speaks the way a real Saudi person would speak to a friend.

She does not speak in Modern Standard Arabic (MSA) as used in news or formal writing.

She does not use heavy dialect words that are not broadly understood across the Arabic-speaking world.

Her language sits between natural conversational Arabic and broadly accessible spoken Arabic.

---

## Core Principle

**Write for the ear, not the eye.**

Every sentence should be heard, not read.

Read every script out loud before considering it finished.

If a sentence sounds unnatural when spoken, rewrite it.

---

## Sentence Structure

* Keep sentences short.
* One idea per sentence.
* Let punctuation create breathing room.
* Avoid long compound sentences connected by many conjunctions.

**Good:**
```
المكان كان هادي. ما توقعت إن فيه هالتاريخ. لكن لما بدأت أسأل، تغيّر كل شيء.
```

**Avoid:**
```
المكان كان هادياً للغاية وما كنت أتوقع إن فيه هذا التاريخ العريق ولكن عندما بدأت بالاستفسار من السكان المحليين تغيّرت نظرتي تغيّراً جذرياً.
```

---

## Language Register

| Use                        | Avoid                          |
|----------------------------|-------------------------------|
| Conversational Arabic      | Modern Standard Arabic (MSA)  |
| Natural contractions       | Formal conjunctions            |
| Everyday vocabulary        | Academic or literary terms     |
| Direct address ("أنتِ")   | Impersonal or third-person style|
| Short rhetorical questions | Long explanatory paragraphs    |

---

## Opening Lines

The first line of every episode must pull the viewer in immediately.

It should create curiosity, establish place, or start mid-thought.

**Good openings:**
```
وصلت المدينة الساعة خمسة الصبح. ما كان فيه أحد.
```
```
سألت نفسي: إيش يخلي الناس تسكن مكان زي هذا لألف سنة؟
```
```
أول ما شفت الحارة، ما عجبتني. لكن بعد ساعة، ما أبيت أطلع.
```

**Avoid:**
```
أهلاً وسهلاً، أنا سارة، ونحن اليوم في مدينة...
```

---

## Closing Lines

The closing should leave something behind — a thought, a feeling, a question.

It should not summarize the episode. It should complete it.

**Good closings:**
```
الأماكن القديمة ما تموت. تنتظر بس لحد يسأل عنها.
```
```
ما أدري متى أرجع. لكن أعرف إني راح أرجع.
```

---

## Rhetorical Questions

Use sparingly but effectively.

A well-placed question invites the viewer into the story.

```
تساءلت: مين اللي بنى هذا؟ وليش هنا بالذات؟
```

---

## Words to Avoid

| Avoid                     | Reason                                 |
|---------------------------|----------------------------------------|
| "أهلاً وسهلاً"            | Overused, presenter-style opening      |
| "في هذا الفيديو سنتعرف"   | Sounds like educational content        |
| "لا تنسوا الاشتراك"       | Breaks the storytelling frame          |
| Excessively formal verbs  | Sounds like a written essay            |
| English words unnecessarily| Breaks voice consistency              |

---

## Numbers and Dates

Speak numbers naturally as they would be said aloud:
* Use conversational forms: "ألف وأربعمية" not "١٤٠٠"
* Historical dates should be given context: "من أكثر من ألف سنة"

---

## Script Review Checklist

Before finalizing any script:

- [ ] Every sentence is short enough to say in one breath or less
- [ ] No sentence sounds like it was written for reading
- [ ] Opening line creates immediate curiosity
- [ ] Closing line completes the story, does not summarize
- [ ] No formal Arabic news-style phrasing
- [ ] Script was read aloud at least once
